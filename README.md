# praxis-setup (öffentlich)

Client-facing Erklärseite für das **Innovation-Bridge-Praxis-Setup**: wie eine
Solo-Praxis von der Beratung über den Bau bis zur Befähigung begleitet wird —
transparent, gefördert, EU-AI-Act-bewusst.

Öffentlich teilbar (GitHub Pages). **Bewusst ohne** interne Preise, Kundennamen
oder interne Notizen — die leben im privaten internen Konzept (`ib/praxis-setup/`).

## Inhalt

| Datei | Was |
|---|---|
| `index.html` | Die Erklärseite: drei Phasen, Übergabe-Logik, End-Nutzer-Loop, EU AI Act / Trust, BAFA, Ablauf, System-Diagramm, CTA |
| `styles.css` | Stylesheet (DM Sans + Geist Mono, warmes Papier) |

Static, kein Build. `noindex` gesetzt (Entwurf, nur per Link teilen).

## Was hier NICHT rein darf

- Preise / Honorare
- Kundennamen oder kundenspezifische Scopes (PII)
- Interne Strategie- oder BAFA-Verifizierungs-Notizen
- Förder-Zusicherungen oder „AI-Act-zertifiziert"-Claims

Personalisierte Angebote gehen über einen privaten Kanal (pwd-Link / PDF), nicht
über diese öffentliche Seite.
